Project Overview

This project involved developing and validating an Anti-lock Braking System (ABS) model using MATLAB and Simulink. The goal was to optimize braking performance under varying driving conditions.

Tools Used

  • MATLAB
  • Simulink

Key Contributions

  • Conducted 50+ test runs to evaluate ABS performance.
  • Improved braking response time by 15% under wet conditions.
  • Integrated Simulink model with MATLAB code for data analysis.
